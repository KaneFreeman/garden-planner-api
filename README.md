# Garden Planner API

Server API for [Garden Planner](https://github.com/KaneFreeman/garden-planner).

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Build

```bash
# production build
$ yarn build
```

## Docker

```bash
# production build
$ yarn docker:up

# restart/rebuild production build
$ yarn docker:restart
```
